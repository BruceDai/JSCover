Contribute

Please feel free to contribute - see the To-Do list and github issues. The difficulty level for contributing patches varies depending on the area.

Rhino bugs can be posted there, but please notify us to avoid duplication. If the report is urgent it can be manually patched and included in JSCover which already uses a forked version of Rhino (there is a pull request to merge this fork back to Rhino trunk).
